# 如何高效？
1. 够懒简化重复性操作，如git命令alias来简化
2. Dash、Alfred、稀土掘金
3. 时间安排方式，日历上计划好任务，没完成产生压力。可预留空闲时间应付突发事件
4. 当面过去沟通虽然高效但会打断对方的节奏。用QQ、微信或邮件一次性说清楚，留有记录方便复查核对

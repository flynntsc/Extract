> 面试的时候，面试官经常会问到，之前工作中遇到什么难点，怎么解决的？然后我个人觉得能解决掉的问题都不是什么难点，所以比较困惑，不知道怎么回答？

不可能完全没有遇到问题，没有=你接触的还太浅了。考察你定位问题，解决问题的能力。一般不建议说没遇到问题，很不好。也不仅限于技术，也可以包括非技术相关的，比如在产品、设计、沟通、流程等上面相关的。

参与社区、开源项目、自己博客或敢与别人吵架,能促进自己真正去掌握知识。
